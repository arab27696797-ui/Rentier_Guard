# Шаблон договора аренды жилого помещения

## Файл: contract_residential.docx

Это описание структуры шаблона. Для использования создайте реальный DOCX файл с указанными переменными.

---

## Структура документа

```
ДОГОВОР АРЕНДЫ [тип помещения]

№ [номер] от [дата]

1. ПРЕДМЕТ ДОГОВОРА
   Описание объекта аренды

2. СРОК ДЕЙСТВИЯ ДОГОВОРА
   Даты начала и окончания

3. АРЕНДНАЯ ПЛАТА И ПОРЯДОК РАСЧЕТОВ
   Сумма, способ и сроки оплаты

4. ПРАВА И ОБЯЗАННОСТИ СТОРОН
   Обязанности арендодателя и арендатора

5. ОТВЕТСТВЕННОСТЬ СТОРОН
   Условия ответственности

6. ЗАКЛЮЧИТЕЛЬНЫЕ ПОЛОЖЕНИЯ
   Прочие условия

7. ПОДПИСИ СТОРОН
   Место для подписей
```

## Пример использования в шаблоне

```
ДОГОВОР АРЕНДЫ {{propertyType}}

№ {{contractNumber}} от {{contractDate}}

г. Москва                                    {{currentDate}}

Иванов Иван Иванович, именуемый в дальнейшем "Арендодатель", 
с одной стороны, и Петров Петр Петрович, именуемый в дальнейшем 
"Арендатор", с другой стороны, совместно именуемые "Стороны", 
заключили настоящий договор о нижеследующем:

1. ПРЕДМЕТ ДОГОВОРА

1.1. Арендодатель предоставляет Арендатору во временное владение 
и пользование {{propertyType}}, расположенное по адресу: 
{{propertyAddress}}.

1.2. Общая площадь помещения составляет {{propertyTotalArea}} кв.м.

3. АРЕНДНАЯ ПЛАТА

3.1. Арендная плата составляет {{monthlyRent}} ({{monthlyRentWords}}) 
в месяц.
```

---

## Создание шаблона

1. Создайте документ в Microsoft Word / LibreOffice Writer
2. Разместите переменные в формате `{{variableName}}`
3. Сохраните как `contract_residential.docx`
4. Поместите в папку `assets/templates/`

---

## Примечания

- Все переменные чувствительны к регистру
- Необязательные переменные можно оставить пустыми
- Для таблиц используйте синтаксис docx-templates
- Поддерживается условная логика (IF/ELSE)
