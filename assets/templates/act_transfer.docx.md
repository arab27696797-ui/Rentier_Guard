# Шаблон акта приема-передачи / возврата

## Файл: act_transfer.docx

Это описание структуры шаблона. Для использования создайте реальный DOCX файл с указанными переменными.

---

## Переменные шаблона

### Данные договора
| Переменная | Описание | Пример |
|------------|----------|--------|
| `{{contractNumber}}` | Номер договора | 123 |
| `{{contractDate}}` | Дата заключения договора | 01.01.2024 |

### Тип акта
| Переменная | Описание | Пример |
|------------|----------|--------|
| `{{actType}}` | Тип акта (приема/возврата) | приема-передачи |

### Данные объекта
| Переменная | Описание | Пример |
|------------|----------|--------|
| `{{propertyAddress}}` | Полный адрес объекта | г. Москва, ул. Арбат, д. 5, кв. 10 |
| `{{propertyCadastral}}` | Кадастровый номер | 77:01:0001234:5678 |
| `{{propertyType}}` | Тип объекта | квартира |
| `{{propertyTotalArea}}` | Общая площадь | 45.5 |

### Стороны
| Переменная | Описание | Пример |
|------------|----------|--------|
| `{{landlordName}}` | ФИО арендодателя | Иванов Иван Иванович |
| `{{tenantName}}` | ФИО арендатора | Петров Петр Петрович |

### Инвентарь (таблица)
| Переменная | Описание | Тип |
|------------|----------|-----|
| `{{inventoryTable}}` | Таблица с инвентарем | Table |
| `{{hasInventory}}` | Наличие инвентаря (true/false) | Boolean |
| `{{inventoryCount}}` | Количество позиций | Number |

### Даты
| Переменная | Описание | Пример |
|------------|----------|--------|
| `{{currentDate}}` | Текущая дата | 01.01.2024 |
| `{{transferDate}}` | Дата передачи/возврата | 01.02.2024 |

---

## Структура таблицы инвентаря

Таблица `{{inventoryTable}}` должна содержать следующие столбцы:

| № | Наименование | Количество | Состояние | Примечание |
|---|--------------|------------|-----------|------------|
| 1 | Холодильник | 1 | Новое | Работает исправно |
| 2 | Стиральная машина | 1 | Б/у | - |
| 3 | Микроволновка | 1 | Новое | - |

## Структура документа

```
АКТ [приема-передачи/возврата] [тип помещения]

к Договору аренды № [номер] от [дата]

Настоящий акт составлен "[день]" [месяц] [год] г.
между:

[Арендодатель] — с одной стороны,
[Арендатор] — с другой стороны,

совместно именуемые "Стороны".

Стороны составили настоящий акт о том, что:

1. [Передано/Возвращено] [тип помещения], расположенное 
   по адресу: [адрес].

2. Передано следующее имущество (см. таблицу):
   [Таблица инвентаря]

3. Стороны претензий друг к другу не имеют.

Подписи сторон:

_________________ / _________________
Арендодатель

_________________ / _________________
Арендатор
```

## Пример использования в шаблоне

```
АКТ {{actType}} {{propertyType}}

к Договору аренды № {{contractNumber}} от {{contractDate}}

Настоящий акт составлен "{{currentDay}}" {{currentMonth}} {{currentYear}} г.
между:

{{landlordName}}, именуемый в дальнейшем "Арендодатель", с одной стороны,
и {{tenantName}}, именуемый в дальнейшем "Арендатор", с другой стороны,
совместно именуемые "Стороны".

Мы, нижеподписавшиеся, составили настоящий акт о том, что:

1. {{actType}} {{propertyType}}, расположенное по адресу: 
   {{propertyAddress}}, общей площадью {{propertyTotalArea}} кв.м.

2. Передано следующее имущество:

{{#if hasInventory}}
| № | Наименование | Кол-во | Состояние | Примечание |
{{#each inventoryTable.rows}}
| {{@index}} | {{this.1}} | {{this.2}} | {{this.3}} | {{this.4}} |
{{/each}}
{{else}}
Имущество отсутствует.
{{/if}}

3. Стороны претензий друг к другу не имеют.

Подписи:

_________________ / {{landlordName}}
(Арендодатель)

_________________ / {{tenantName}}
(Арендатор)
```

## Синтаксис таблицы в docx-templates

Для создания динамической таблицы используйте:

```
{{#if hasInventory}}
+------+----------------+----------+-----------+------------+
|  №   | Наименование   | Кол-во   | Состояние | Примечание |
+======+================+==========+===========+============+
{{#each inventory}}
| {{@index}} | {{name}} | {{qty}} | {{cond}} | {{note}} |
+------+----------------+----------+-----------+------------+
{{/each}}
{{/if}}
```

---

## Создание шаблона

1. Создайте документ в Microsoft Word / LibreOffice Writer
2. Разместите переменные в формате `{{variableName}}`
3. Добавьте таблицу для инвентаря
4. Сохраните как `act_transfer.docx`
5. Поместите в папку `assets/templates/`

---

## Различия актов

### Акт приема-передачи (начало аренды)
- Передача от Арендодателя к Арендатору
- Получение залога
- Фиксация начального состояния

### Акт возврата (окончание аренды)
- Возврат от Арендатора к Арендодателю
- Возврат залога
- Проверка состояния имущества
- Фиксация повреждений (если есть)

---

## Примечания

- Таблица инвентаря опциональна
- Для акта возврата используйте тот же шаблон с `actType = "возврата"`
- Рекомендуется фотографировать состояние при передаче
- Подписи обеих сторон обязательны
