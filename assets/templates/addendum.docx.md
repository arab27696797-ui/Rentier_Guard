# Шаблон дополнительного соглашения

## Файл: addendum.docx

Это описание структуры шаблона. Для использования создайте реальный DOCX файл с указанными переменными.

---

## Переменные шаблона

### Данные договора
| Переменная | Описание | Пример |
|------------|----------|--------|
| `{{contractNumber}}` | Номер договора | 123 |
| `{{contractDate}}` | Дата заключения договора | 01.01.2024 |

### Данные изменения
| Переменная | Описание | Пример |
|------------|----------|--------|
| `{{changeType}}` | Тип изменения | изменение арендной платы |
| `{{oldValue}}` | Старое значение | 50 000 рублей |
| `{{newValue}}` | Новое значение | 55 000 рублей |
| `{{changeReason}}` | Причина изменения | индексация согласно инфляции |

### Стороны
| Переменная | Описание | Пример |
|------------|----------|--------|
| `{{landlordName}}` | ФИО арендодателя | Иванов Иван Иванович |
| `{{tenantName}}` | ФИО арендатора | Петров Петр Петрович |

### Даты
| Переменная | Описание | Пример |
|------------|----------|--------|
| `{{currentDate}}` | Текущая дата | 01.06.2024 |
| `{{addendumDate}}` | Дата допсоглашения | 01.06.2024 |
| `{{effectiveDate}}` | Дата вступления в силу | 01.07.2024 |

---

## Типы изменений (примеры)

| Тип изменения | oldValue | newValue |
|---------------|----------|----------|
| изменение арендной платы | 50 000 руб. | 55 000 руб. |
| изменение срока аренды | 01.02.2025 | 01.08.2025 |
| изменение способа оплаты | наличные | банковский перевод |
| изменение порядка оплаты | 5-го числа | 10-го числа |
| добавление условий | - | разрешено содержание кошки |
| изменение залога | 50 000 руб. | 55 000 руб. |
| изменение состава проживающих | 2 человека | 3 человека |

## Структура документа

```
ДОПОЛНИТЕЛЬНОЕ СОГЛАШЕНИЕ

к Договору аренды № [номер] от [дата]

г. [город]                                    [дата]

[Арендодатель], именуемый в дальнейшем "Арендодатель", 
с одной стороны, и [Арендатор], именуемый в дальнейшем 
"Арендатор", с другой стороны, совместно именуемые "Стороны", 
заключили настоящее дополнительное соглашение о нижеследующем:

1. Внести следующие изменения в Договор:
   
   1.1. [Тип изменения]:
        - было: [старое значение]
        - стало: [новое значение]
   
   1.2. [Причина изменения]

2. Настоящее дополнительное соглашение вступает в силу 
   с [дата вступления в силу].

3. Остальные условия Договора остаются без изменений.

4. Настоящее дополнительное соглашение составлено в двух 
   экземплярах, имеющих одинаковую юридическую силу.

Подписи сторон:

_________________ / _________________
Арендодатель

_________________ / _________________
Арендатор
```

## Пример использования в шаблоне

```
ДОПОЛНИТЕЛЬНОЕ СОГЛАШЕНИЕ

к Договору аренды № {{contractNumber}} от {{contractDate}}

г. Москва                                    {{addendumDate}}

{{landlordName}}, именуемый в дальнейшем "Арендодатель", 
с одной стороны, и {{tenantName}}, именуемый в дальнейшем 
"Арендатор", с другой стороны, совместно именуемые "Стороны", 
заключили настоящее дополнительное соглашение о нижеследующем:

1. Внести следующие изменения в Договор:

   1.1. {{changeType}}:
        - было: {{oldValue}}
        - стало: {{newValue}}

   {{#if changeReason}}
   1.2. Причина изменения: {{changeReason}}
   {{/if}}

2. Настоящее дополнительное соглашение вступает в силу 
   с {{effectiveDate}}.

3. Остальные условия Договора остаются без изменений.

4. Настоящее дополнительное соглашение составлено в двух 
   экземплярах, имеющих одинаковую юридическую силу.

Подписи сторон:

_________________ / {{landlordName}}
(Арендодатель)

_________________ / {{tenantName}}
(Арендатор)
```

## Несколько изменений в одном допсоглашении

Если нужно внести несколько изменений:

```
1. Внести следующие изменения в Договор:

   1.1. Изменение арендной платы:
        - было: 50 000 руб.
        - стало: 55 000 руб.
   
   1.2. Изменение дня оплаты:
        - было: 5-го числа
        - стало: 10-го числа
   
   1.3. Изменение срока договора:
        - было: до 01.02.2025
        - стало: до 01.08.2025
```

## Создание шаблона

1. Создайте документ в Microsoft Word / LibreOffice Writer
2. Разместите переменные в формате `{{variableName}}`
3. Сохраните как `addendum.docx`
4. Поместите в папку `assets/templates/`

---

## Рекомендации

### Когда составлять допсоглашение:
- ✅ Изменение арендной платы
- ✅ Продление срока договора
- ✅ Изменение условий оплаты
- ✅ Добавление/изменение условий
- ✅ Смена арендатора (с согласия)

### Когда заключать новый договор:
- ❌ Смена объекта аренды
- ❌ Полная смена сторон
- ❌ Кардинальные изменения условий
- ❌ Истечение срока старого договора

---

## Примечания

- Допсоглашение имеет ту же юридическую силу, что и основной договор
- Стороны должны подписать оба экземпляра
- Рекомендуется регистрировать изменения (если требуется законом)
- Сохраняйте копии всех допсоглашений
